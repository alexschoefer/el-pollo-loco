**El Pollo Loco** is a classic 2D jump-and-run browser game built using HTML, CSS, and JavaScript. 
The player controls a character navigating through animated levels, collecting items, defeating enemies, and trying to reach the final boss: the crazy chicken!
The project focuses on object-oriented programming, dynamic rendering, collision detection, and smooth animations ‚Äî all implemented without any game engine.

## Technologies Used

- JavaScript (OOP, Canvas, event handling)
- HTML
- CSS

## Game Features

- Animated jumps, throws, movements, and enemy interactions
- Collision detection for enemies, objects, and projectiles
- Collect items (bottles, coins) to use as weapons or score points
- Energy and status bars for the player and enemies
- Boss fight against *El Pollo Loco*
- Sound effects and background animations
- Responsive gameplay in the browser

## How to Play

- Arrow keys to move and jump
- Space bar (or assigned key) to throw projectiles
- Reach the end of the map and defeat the boss
